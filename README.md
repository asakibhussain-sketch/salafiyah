# Salafiyah — Find Peace. Stay Connected. 🕌

Salafiyah is a modern, responsive, and feature-rich Progressive Web Application (PWA) designed to be your spiritual companion. Built as a Single Page Application (SPA), it provides users with tools to read the Quran, track prayers, count Dhikr with AI guidance, study authentic Duas, and consult an AI Imam assistant.

---

## 🌟 Features

*   **Interactive Quran & Mushaf**: Beautiful, readable interfaces for engaging with the Holy Qur'an.
*   **AI Ask Imam Assistant**: A smart, compassionate chat assistant powered by `llama-3.3-70b-versatile` (via Groq) to help answer questions based on authentic Islamic sources.
*   **Contextual AI Tasbih**: Generate custom, context-specific Dhikr recommendations based on your current spiritual state or intention.
*   **Spiritual Tracker & Insights**: Keep track of your daily prayers and Dhikr counts to build consistent habits.
*   **Authentic Duas Library**: Access a curated collection of supplications.
*   **Premium Custom Themes**: Includes beautiful, curated themes (Light, Dark, and spiritual variations) with glassmorphism styling and smooth animations.
*   **PWA Offline Ready**: Configured with a service worker (`sw.js`) and web app manifest (`manifest.json`) for installing the app directly onto your mobile device.
*   **Secure SQLite Backend**: FastAPI server with secure OTP-based email sign-up/password recovery, and SQLite data syncing.
*   **Automated Mobile Testing Tunnel**: Integrated `ngrok` setup to easily tunnel the application for testing on physical mobile devices.

---

## 🛠️ Architecture

*   **Frontend**: Vanilla HTML5, CSS3 (Custom Variables, glassmorphic styling, custom typography via Google Fonts), and JavaScript (SPA router, Service Worker).
*   **Backend**: Python, FastAPI, SQLite (database persistence), and `httpx` (AI API calls).
*   **AI Integration**: Groq API (`llama-3.3-70b-versatile`).

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have Python 3.8+ installed on your system.

### 2. Installation
Clone this repository and navigate to the project directory:
```bash
git clone <your-repo-url>
cd salafiyah
```

Install the required Python dependencies:
```bash
pip install -r requirements.txt
```

### 3. Configuration
Create a `.env` file in the root directory and add your credentials:
```env
GROQ_API_KEY=your_groq_api_key_here
NGROK_AUTH_TOKEN=your_ngrok_auth_token_here
```

### 4. Running the Application

#### Option A: Running locally
To start the FastAPI backend server locally on port 8000:
```bash
python main.py
```
Open your browser and visit: `http://localhost:8000`

#### Option B: Running with a public ngrok tunnel (Mobile Testing)
To launch the FastAPI server and automatically expose it through a secure `ngrok` tunnel (allowing you to access the app on your phone):
```bash
python start_with_tunnel.py
```
Copy the public `https://...ngrok-free.app` URL printed in the console and load it on any mobile device.

```text
├── salafiyah.db            # SQLite database (autogenerated on first run)
├── requirements.txt        # Backend dependencies
└── .gitignore              # Standard git exclusion rules (.env, database, cache)
```
